# Beautyacticle
> Share the most beautiful words

> 做一个好用的文字APP

直接下载：http://fir.im/beautyacticle
或者：http://www.pgyer.com/beautyacticle

- Material Design
- 精选好图好文，支持本地缓存
- 文章支持MarkDown排版
- 文艺范分享图文

###后续
